To Run:
1. Install Node - brew install node
2. Install bower - npm install -g bower
3. Run npm update
4. Run bower install
5. Run the app as 
	node app.js

You can observe the endpoint output appearing in the console from the Twitter API.


You can login at 127.0.0.1:3000 to view the page displaying the Twubric scores.
**The UI has a current refresh issue that I hope to fix soon so if possible please do a manual refresh of the page and it should display the names and scores.
